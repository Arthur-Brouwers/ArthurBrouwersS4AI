We have already established that the variable age lacks quite some values and now we want to see if that is somehow related to values of other variables in the same observation. We are especially interested in the relations between age and the variables pclass, sex and embarked.

pclass is stored as an integer, but it is actually a nominal (categorical) variable. Because chartting engines are often lead in chosing optimal visuals by data types, lets convert pclass to its proper type.

In [ ]:
df['pclass'] = df['pclass'].astype('category')

Let's first investigate the relationship between age and pclass. We're mainly interested in the ratio of nulls and non-nulls per category key:

As ypu can see, this graph doesn't mirror reality: it just connect 2 subsequent days with sales, assuming that possible days between had sales values on the connecting line. That is of course wrong: missing days in the data set had no sales at all! We can make that obvious by interpolating missing days with zero sales value using a (daily) resampler:

In [ ]:
(
    sales
    .query('"20190101" <= orderdate <= "20191231"')
    .set_index('orderdate')
    .resample('D')['linetotal']
    .sum()
    .plot.line()
)
Out[ ]:
<Axes: xlabel='orderdate'>

You can resample a time series database on various sample frequencies as demonstrated in the following plot:

In [ ]:
frequencies = ['M', 'W', 'D']

for i, freq in enumerate(frequencies):
    dysales.resample(freq).linetotal.sum().plot.line(ax=ax[i])

We use cumsum() for cumulative sums. It's a strange function that depends for its order on the index.

In [ ]:
(
    sales
    .assign(year=sales.orderdate.dt.year , month=sales.orderdate.dt.month)
    .groupby(['year', 'month'])['linetotal']
    .sum()
    .groupby(level=0)   # we want to group on year: it's the first (0) level of the index
    .cumsum()
    .unstack(level=0)
    .plot.line()
)

import pandas as pd
import matplotlib.pyplot as plt

# Ensure that the 'orderdate' column is in datetime format
sales['orderdate'] = pd.to_datetime(sales['orderdate'])

# Extract the year and weekday from the 'orderdate' column
sales['Year'] = sales['orderdate'].dt.year
sales['Weekday'] = sales['orderdate'].dt.weekday  # Monday is 0, Sunday is 6

# Calculate average daily sales for each weekday in each year
avg_daily_sales = sales.groupby(['Year', 'Weekday'])['linetotal'].mean().unstack()

# Define the weekdays for labeling
weekdays = ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday', 'Sunday']

# Create a line chart
plt.figure(figsize=(12, 6))
for weekday in range(7):
    plt.plot(avg_daily_sales.index, avg_daily_sales[weekday], label=weekdays[weekday])

# Label the chart
plt.title('Average Daily Sales by Weekday Over the Years')
plt.xlabel('Year')
plt.ylabel('Average Daily Sales')
plt.legend()

# Show the chart
plt.show()
