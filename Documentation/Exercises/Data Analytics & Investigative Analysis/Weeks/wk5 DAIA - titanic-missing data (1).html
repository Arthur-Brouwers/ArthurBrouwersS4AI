Missing data in the Titanic dataset¶

1. Introduction¶

In this titanic example we will focus on how to find, label and deal with missing data. Make sure you have studies the study materials in Canvas about the various types of missing data and how to handle these kind of missing data.

Each instance in the dataset has the following form:

 "1","1st",1,"Allen, Miss Elisabeth Walton",29.0000,"Southampton","St Louis, MO","B-5","24160 L221","2","female"

Note that the raw data consists largely of strings. To apply machine learning algo's these strings have to be converted to numerical data first (at least the columns that are of interest)!

2. Data collection¶

Pandas is a Python module that works with the so-called dataframe concept (rows are observations, columns refer to the features). A dataframe is essentially a two-dimensional labeled data structure where each column represent a feature and each row represents an observation.

More details, see: https://www.kaggle.com/c/titanic/details/getting-started-with-python-ii

4. Data Preparation¶

4.1. Deal with Missing Values¶

You have found some (indications, clues for) missing data. Under 'Data Understanding' you have looked for reasons why and perhaps also found some relationships with other values or features. That helps you to 'label' the type of missing data. Now it's time to deal with the missing values. It could be a requirement of the machine learning model that it needs a complete set of values for a feature. There are several options as you know. Deleting rows can reduce missing values, but it also reduces the total amount of data... Another way is to fill it in with guesses. You will be introducing some noise into a model, but if you canmake reasonable guesses, some of them should be close to the historical truth (whatever it was...), and the overall predictive power of the model might still make a better model than before.

import pandas as pd
import random

# Calculate the number of missing values and the number of unique 'boat' categories
missing_count = df_titanic['boat'].isnull().sum()
unique_boats = df_titanic['boat'].nunique()

# Determine how many missing values should be imputed for each 'boat' category
impute_per_category = missing_count // unique_boats

# Create a DataFrame with missing 'boat' values
missing_boat_df = df_titanic[df_titanic['boat'].isna()]

# Initialize an empty list to store the imputed 'boat' values
imputed_boats = []

# Iterate over unique 'boat' categories
for boat_category in df_titanic['boat'].unique():
    # Check if there are any missing values in the 'boat' column for this category
    if missing_boat_df[missing_boat_df['boat'].isna()].empty:
        continue
    
    # Select 'impute_per_category' random rows for this 'boat' category
    random_rows = missing_boat_df.sample(impute_per_category, random_state=42)
    
    # Assign the 'boat' category to the selected rows
    random_rows['boat'] = boat_category
    
    # Add the selected rows to the list of imputed 'boat' values
    imputed_boats.append(random_rows)

# Concatenate the imputed 'boat' values back to the original DataFrame
imputed_df = pd.concat([df_titanic, *imputed_boats], ignore_index=True)
